## UX Neurodivergent Challenges Reflection

## Reflection

**How can Focus Bear’s design choices make tasks easier for neurodivergent users?**

    Focus Bear’s best design choice is linear progression. By showing only one habit or one task at a time, it removes the need for the user to choose what to do next. It essentially acts as a tunnel, guiding the user from start to finish without letting them get sidetracked by a messy dashboard. This is especially helpful for users who might feel overwhelmed by too many options or a cluttered interface. The simplicity of the design allows users to focus on one thing at a time, which can reduce anxiety and increase productivity.

**If a feature is technically useful but cognitively overwhelming, how should it be redesigned?**

    It should be layered. We can use progressive disclosure, which means we hide the advanced settings or extra info behind a "More" button or a separate tab. The main screen should always stay clean and simple, while the technical "power features" stay out of the way until the user actually goes looking for them. This way, we can keep the interface clean and user-friendly for everyone, while still providing access to the more complex features for those who need them. It’s all about giving users control over how much information they want to see at once, which can help reduce cognitive overload and make the app more accessible to a wider range of users.

**What are some small UX adjustments that can significantly improve user experience for neurodivergent users?**

    Some small adjustments include using friendly, clear language like "You're doing great!" instead of robotic error messages. Another UX adjustment could be using high-contrast colors only for the most important action (like the "Done" button). This way, users can easily identify what they need to click without feeling overwhelmed by too many bright colors. We can also add buffer times like giving the user a 10-second warning before a block starts so they don't feel jolted out of what they were doing. This can help users mentally prepare for the next task and reduce feelings of being rushed or overwhelmed. These small changes can make a big difference in creating a more inclusive and supportive user experience for neurodivergent users.
